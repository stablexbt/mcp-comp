# Product Hunt Launch

## Tagline
Ship compliant AI agent payments in 10 minutes

## Description
REM Payment Server is an MCP-native payment server that lets AI agents make compliant payments. It wraps payment execution with built-in KYT (Know Your Transaction), Travel Rule (IVMS 101), and a configurable policy engine.

### Why REM?
AI agents are starting to make payments, but most implementations have zero compliance infrastructure. REM solves this by providing:
- Transaction screening before execution
- Policy controls for what agents can buy
- Audit trail for every payment decision
- x402 compliance out of the box

### How It Works
1. Add REM to your MCP config
2. Your agent gets payment tools automatically
3. Every payment goes through compliance checks
4. Clear pass/fail with reasons

### Open Source
The core MCP server is open source under MIT license. Use it for free, forever.

## Maker Comment
We built REM after seeing how many teams were shipping agent payments with hardcoded API keys and no compliance layer. If you're building AI agents that need to spend money, this is the infrastructure you need.
